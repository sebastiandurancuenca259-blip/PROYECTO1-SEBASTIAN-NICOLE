# 🌐 Hoja de Vida en Línea

## 📌 Descripción

Este proyecto consiste en el desarrollo de una hoja de vida (CV) digital presentada como una página web estática, desplegada en internet mediante GitHub. Su propósito es mostrar de forma moderna, ordenada y accesible la información personal, académica y las habilidades de los estudiantes.

La plataforma permite visualizar el perfil profesional desde cualquier dispositivo con acceso a internet, ofreciendo una alternativa innovadora frente a los currículums tradicionales en formato físico o PDF.

El diseño está enfocado en la claridad, la estética visual y la experiencia del usuario, facilitando una navegación intuitiva y agradable.

## 🎯 Objetivo

Demostrar habilidades en desarrollo web front-end mediante la creación de un portafolio digital funcional, al mismo tiempo que se construye una presencia profesional en línea accesible para docentes, reclutadores y público en general.
